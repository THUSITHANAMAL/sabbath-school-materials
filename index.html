How to add your PDF files:

1. Create a folder named pdfs in your project directory
2. Inside pdfs, create three folders: grade1-4, grade5-7, and grade8-9
3. Place your PDF files in the appropriate grade folders
4. Name your PDFs like: Q1-L1.pdf, Q1-L2.pdf, etc.
5. Update the lesson titles in the HTML to match your actual lessons
6. Push everything to your GitHub repository
